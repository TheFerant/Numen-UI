## Introduction

### About

**numen UI** is an elegant and beautiful React component library with Geek style,
originating from [Vercel](https://vercel.com?utm_source=numen-ui&utm_campaign=oss) design system.

With numen's unique advantages in text layout, optimization of light and dark lines, and scale components,
you can easily create clean and powerful personal sites, or quickly build modern web applications based on highly customizable design system.

Get start on numen with the [guide](/en-us/guide/installation),
or view full [project examples](https://github.com/numen-org/numen-ui/tree/master/examples).
